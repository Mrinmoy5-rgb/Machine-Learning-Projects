# 🚀 INTERNPE Machine Learning Internship

Welcome to the **INTERNPE Machine Learning Internship Projects** repository.

This folder contains machine learning projects developed during the **INTERNPE Internship**, focusing on **data preprocessing, model training, and prediction systems** using Python and Scikit-Learn.

---

## 📂 Projects Included

### 🩺 Diabetes Prediction

A **classification model** that predicts whether a person is diabetic based on medical features.

🔍 **Key Concepts Used**

* Data Cleaning
* Feature Scaling
* Train-Test Split
* Support Vector Machine (SVM)
* Model Evaluation

📁 Folder
`Diabetes_Prediction`

---

### 🚗 Car Price Prediction

A **regression model** that estimates the selling price of a car based on features such as company, year, fuel type, and kilometers driven.

🔍 **Key Concepts Used**

* Data Preprocessing
* Feature Encoding
* Linear Regression
* Random Forest Regressor
* Model Comparison

📁 Folder
`Car_Price_Prediction`

---

### 🧠 Breast Cancer Prediction

A **classification model** that predicts whether a tumor is **Malignant (Cancerous)** or **Benign (Non-cancerous)** using medical diagnostic data.

🔍 **Key Concepts Used**

* Data Cleaning & Preprocessing
* Feature Scaling (StandardScaler)
* Train-Test Split
* K-Nearest Neighbors (KNN)
* Hyperparameter Tuning (GridSearchCV)
* Model Evaluation (Confusion Matrix, Precision, Recall)

📁 Folder
`breast-cancer-prediction`
